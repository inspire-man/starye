## 功能规划

### 1. Aria2 RPC 集成 🔥 高优先级

**需求描述**：
集成 Aria2 下载管理器，实现从浏览器直接发起下载任务。

**功能详情**：
- **Aria2 连接管理**
  - 配置 Aria2 RPC 地址（WebSocket 或 HTTP）
  - 连接状态检测和自动重连
  - 支持密钥认证（Secret Token）
  
- **下载任务管理**
  - 一键添加磁链到 Aria2
  - 批量添加多个磁链
  - 查看下载进度（实时）
  - 暂停/恢复/删除任务
  
- **任务状态同步**
  - WebSocket 实时推送下载进度
  - 下载完成通知
  - 自动更新下载列表状态

**技术方案**：
```typescript
// 示例 API 设计
interface Aria2Config {
  rpcUrl: string          // ws://localhost:6800/jsonrpc
  secret?: string         // RPC 密钥
  autoConnect: boolean    // 自动连接
}

interface DownloadTask {
  gid: string            // Aria2 任务 ID
  status: 'active' | 'waiting' | 'paused' | 'error' | 'complete'
  totalLength: number
  completedLength: number
  downloadSpeed: number
  files: Array<{ path: string, length: number }>
}

// Composable
const { 
  connect, 
  addUri, 
  getTasks, 
  pauseTask, 
  removeTask 
} = useAria2()
```

**实现优先级**：🔥 高
**预估工作量**：3-5 天
**依赖**：用户需自行安装 Aria2

### 2. 播放源质量评分系统 ⭐ 中优先级

**需求描述**：
为播放源添加质量评分，帮助用户选择最佳播放源。

**功能详情**：
- **自动评分指标**
  - 画质（4K > 1080P > HD > SD）
  - 文件大小（合理范围内越大越好）
  - 种子活跃度（seeders/leechers）
  - 来源可信度
  
- **用户评分**
  - 用户可对播放源打分（1-5 星）
  - 评分统计和平均分显示
  - 评分趋势分析
  
- **智能推荐**
  - 根据评分自动推荐最佳播放源
  - "推荐"标签高亮显示
  - 考虑用户历史偏好

**UI 设计**：
```
播放源 1 [4K] ⭐ 4.8 (128人评价) 🏆 推荐
  📊 文件: 8.5GB | 🌱 活跃: 256/18
  [复制] [打开] [二维码] [评分]
```

**实现优先级**：⭐ 中
**预估工作量**：2-3 天
**依赖**：后端 API 支持

### 3. 种子健康度检测 🌱 中优先级

**需求描述**：
实时检测磁力链接的健康度，包括 seeders、leechers、下载速度预估。

**功能详情**：
- **实时健康检查**
  - 通过 DHT 网络查询种子信息
  - 显示 seeders/leechers 数量
  - 更新时间显示
  
- **健康度指标**
  - 健康（绿色）：seeders > 50
  - 一般（黄色）：10 < seeders ≤ 50
  - 不佳（红色）：seeders ≤ 10
  - 死链（灰色）：无响应
  
- **批量检测**
  - 后台自动检测所有磁链
  - 缓存检测结果（有效期 1 小时）
  - 定期更新热门影片的磁链健康度

**技术方案**：
- 后端使用 BT tracker 查询
- 前端通过 WebSocket 接收实时更新
- IndexedDB 缓存结果

**实现优先级**：🌱 中
**预估工作量**：4-5 天
**依赖**：后端爬虫支持

### 9. WebTorrent 浏览器内播放 🎬 探索性

**需求描述**：
使用 WebTorrent 技术在浏览器内直接播放磁力链接，无需外部客户端。

**功能详情**：
- **即时播放**
  - 点击磁链即可在浏览器内播放
  - 边下边播，无需等待下载完成
  - 支持拖拽进度条
  
- **播放器集成**
  - 集成 Video.js 或其他播放器
  - 支持字幕加载
  - 画质切换（如果有多个源）
  
- **分享功能**
  - 生成可分享的播放链接
  - P2P 加速（其他观看者互相分享）

**技术方案**：
```typescript
import WebTorrent from 'webtorrent'

const client = new WebTorrent()
client.add(magnetURI, (torrent) => {
  const file = torrent.files[0]
  file.renderTo('video')
})
```

**实现优先级**：🎬 探索性（需要技术验证）
**预估工作量**：5-7 天
**依赖**：WebTorrent 库、HTTPS 环境
**风险**：浏览器限制、性能问题

## 技术选型

### Aria2 集成
- **库**：`aria2-jsonrpc-client` 或自实现
- **协议**：WebSocket (推荐) 或 HTTP
- **安全**：支持 Secret Token 认证

- **后端**：Python + libtorrent
- **API**：RESTful + WebSocket
- **缓存**：Redis（1 小时 TTL）

- **存储**：Cloudflare KV 或 D1
- **同步**：WebSocket 推送
- **冲突**：CRDT 算法

### WebTorrent（探索性）
- **库**：`webtorrent@^2.0.0`
- **播放器**：Video.js
- **限制**：仅 HTTPS 环境

## 风险和挑战

### 技术风险
- **Aria2 兼容性**：不同版本 API 差异
- **WebTorrent 性能**：大文件播放可能卡顿
- **浏览器限制**：某些功能可能受限

### 产品风险
- **功能复杂度**：功能过多可能影响易用性
- **性能问题**：大量数据可能导致性能下降
- **隐私问题**：云同步需要处理敏感数据

### 缓解措施
- 渐进式开发，及时收集反馈
- 充分的性能测试和优化
- 清晰的隐私政策和数据加密
