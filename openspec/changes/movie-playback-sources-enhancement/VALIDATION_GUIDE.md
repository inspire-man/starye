# 播放源功能验证指南

本文档提供详细的手动验证步骤，确保所有功能在生产环境正常工作。

## 前提条件

- ✅ 代码已部署到 Cloudflare Pages
- ✅ 访问地址：https://starye.org/movie/ (或您的生产域名)
- 📱 建议准备：桌面浏览器 + 移动设备

## 验证清单

### 1. 影片详情页播放源区块 (任务 9.2)

**测试步骤**：
1. 访问任意影片详情页
2. 向下滚动到"播放源"区块
3. 检查以下内容：
   - [ ] 播放源区块正确渲染
   - [ ] 显示源名称、画质标签、类型图标
   - [ ] 磁力链接和在线播放源分别显示
   - [ ] 排序正确（磁力优先，画质从高到低）

**预期结果**：
- 播放源按照 4K > 1080P/HD > 720P > SD 排序
- 磁力链接显示"📋 复制"、"🔗 打开"、"📱 二维码"按钮
- 在线播放显示"▶️ 播放"按钮

---

### 2. 磁链复制功能 (任务 9.3)

**测试步骤**：
1. 点击任意磁力链接的"📋 复制"按钮
2. 打开记事本或文本编辑器
3. 粘贴内容（Ctrl+V 或 右键粘贴）

**预期结果**：
- [ ] 显示成功提示："磁链已复制到剪贴板"
- [ ] 粘贴内容为完整的 magnet:// 链接
- [ ] 在 Chrome、Firefox、Edge 浏览器中均可正常工作

**降级测试**：
- 如果浏览器权限被拒绝，应自动降级到 fallback 方案

---

### 3. Magnet 协议调用 (任务 9.4)

**测试步骤**：
1. 确保电脑已安装 BT 客户端（如 qBittorrent、迅雷等）
2. 点击磁力链接的"🔗 打开"按钮

**预期结果**：
- [ ] 浏览器提示选择应用程序
- [ ] 选择 BT 客户端后，客户端自动打开
- [ ] 磁力链接正确传递到 BT 客户端

**注意**：首次使用可能需要授权浏览器调用外部程序

---

### 4. 批量复制磁链 (任务 9.5)

**测试步骤**：
1. 在播放源区块点击"📋 复制全部磁链"按钮
2. 粘贴到文本编辑器

**预期结果**：
- [ ] 显示提示："已复制 N 个磁力链接"
- [ ] 粘贴内容格式正确，每行一个磁链
- [ ] 每行包含：源名称、画质、磁力链接

**预期格式示例**：
```
磁力链接 4K [4K]
magnet:?xt=urn:btih:...

磁力链接 HD [HD]
magnet:?xt=urn:btih:...
```

## 错误日志监控 (任务 11.7)

### 监控方式

**Cloudflare Pages 日志**：
1. 登录 Cloudflare Dashboard
2. 进入 Pages -> starye-movie
3. 查看 Analytics -> Real-time logs

**浏览器控制台**：
1. 打开开发者工具 (F12)
2. 切换到 Console 标签
3. 检查是否有错误或警告

### 常见问题排查

**问题 1: 复制失败**
- 检查浏览器权限
- 确认 HTTPS 环境
- 验证 fallback 方案

**问题 2: localStorage 超限**
- 清理浏览器缓存
- 检查其他网站存储占用

**问题 3: 二维码不显示**
- 检查 qrcode.vue 依赖
- 验证 build 输出
