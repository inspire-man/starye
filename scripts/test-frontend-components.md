## 故障排查

### 常见问题

**Q1: 评分组件不显示？**
- 检查 RatingStars.vue 是否正确导入
- 检查 Player 类型是否包含评分字段
- 查看浏览器控制台是否有错误

**Q2: Aria2 按钮不显示？**
- 检查是否为磁力链接播放源
- 检查 useAria2 composable 是否正确导入
- 验证 aria2Connected 状态

**Q3: 下载任务 Tab 空白？**
- 检查 DownloadTaskPanel 组件是否正确导入
- 检查是否配置了 Aria2
- 验证 Aria2 服务是否运行

**Q4: 评分提交失败？**
- 检查是否已登录
- 查看网络请求是否到达后端
- 检查后端日志是否有错误

**Q5: 页面样式混乱？**
- 检查 Tailwind CSS 是否正确加载
- 检查组件的 scoped style 是否冲突
- 验证深色模式配置
