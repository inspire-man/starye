| Boundary | Description |
|----------|-------------|
| GitHub Actions -> remote D1 | CI 拥有直接改写生产数据库的能力 |
| Reviewer -> destructive migration gate | 人工 ack 是危险 schema 变更的最后防线 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04 | Repudiation | migrations apply without backup | mitigate | apply 前强制 remote export，保留恢复点 |
| T-05-05 | Tampering | destructive SQL lands silently | mitigate | `DROP COLUMN` 等危险变更默认阻塞，需明确 ack |
| T-05-06 | Denial of Service | failed migration leaves service unrecoverable | mitigate | RUNBOOK / MIGRATION.md 写清 restore and forward-fix path |

</threat_model>
