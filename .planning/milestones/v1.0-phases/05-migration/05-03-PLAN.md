<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Worker runtime -> Sentry | 请求/错误元数据会发往外部观测平台 |
| Player error path -> message telemetry | 非 crash 播放错误需要既可观察又避免噪音泛滥 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-07 | Information Disclosure | Sentry event payloads | mitigate | 只发送诊断所需上下文，不把敏感 cookie/session 数据带入事件 |
| T-05-08 | Denial of Service | error noise flood | mitigate | `beforeSend` 先过滤 AbortError/NetworkError/自恢复媒体噪音 |
| T-05-09 | Repudiation | video failure invisible in production | mitigate | Player 错误路径统一发送 message/event，上线后可追踪 |

</threat_model>
