<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| admin route -> remote image hosts | integrity probe 会让服务端主动访问外部 URL，必须严防 SSRF 和无限制网络访问 |
| public route -> Reader | public chapter contract 一旦漂移，Reader 就会在不改前端的情况下出现回归 |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-07-04 | Information Disclosure / Elevation | admin integrity probe | high | mitigate | probe 只允许章节已存 URL，拒绝私网/localhost/非 http(s)，限制超时、并发和重定向 |
| T-07-05 | Tampering | public chapter route | medium | mitigate | 锁住 `images: string[]` 原样返回和 pageNumber 排序，避免在 Phase 7 里引入 rewrite 假设 |
| T-07-06 | Repudiation | `/check` vs `/integrity` semantics | medium | mitigate | 保持 cheap local check 与真实 probe 分离，避免运维误把本地 count 当成真实外链健康结论 |
</threat_model>
