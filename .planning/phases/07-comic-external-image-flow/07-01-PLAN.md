## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| crawler strategy -> admin sync | 章节页 URL 会直接从抓取侧进入数据库，错误标准化会污染整个阅读链路 |
| admin sync -> D1 chapter pages | delete-first 行为若没有前置 guard，会把失败抓取升级成永久数据损失 |
