<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| shared helper -> API/crawler/admin consumers | 如果 helper 只覆盖一半语义，后续仍会继续出现“常量共享了，但逻辑没共享”的假统一 |
| helper direct tests -> runtime regressions | 如果 shared helper 没有 direct tests，后续 drift 只能靠调用点偶然发现 |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-10-01 | Tampering / Drift | shared key/url semantics | high | mitigate | 把 manual key builder、crawler namespace helper、URL classifier 都收进 `@starye/api-types` |
| T-10-02 | Regression | upload route | medium | mitigate | 扩展现有 route tests，证明 shared-helper adoption 不改变对外 key/url contract |
</threat_model>
