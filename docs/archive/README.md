# Docs Archive

这里存放 historical / superseded 文档。

## 何时放进来

- 一次性测试报告
- 已完成任务总结
- 被当前 `RUNBOOK.md`、`.planning/*` 或稳定专题文档取代的旧部署/运维手册
- 不再属于 live docs 的旧实现说明

## 何时不要放进来

- 当前 milestone / phase 执行证据
  - 这些继续留在 [`.planning/phases/`](../../.planning/phases/) 和 [`.planning/milestones/`](../../.planning/milestones/)
- 仍然具有稳定参考价值的专题文档
  - 这些继续留在 [`docs/`](../)
